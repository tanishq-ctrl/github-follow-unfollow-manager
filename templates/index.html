GitHub Token Setup

How to Get Your GitHub Token
  1. Visit GitHub Personal Access Tokens.
  2. Click on Generate new token (or Generate new token (classic) if applicable).
  3. Give your token a descriptive name (e.g., GitHub Automation Tool).
  4. Select the following scopes:
    • user:follow (to follow and unfollow users)
    • read:user (to read your user information)
  5. Click Generate token at the bottom of the page.
  6. Copy your token immediately and paste it in the box above.
